## üìã Migration Notes

### Upgrading from 1.x to 2.0

If you have a previous installation:

1. **Backup your data**:
   ```bash
   cp app.db app.db.backup
   ```

2. **Update to new version**:
   ```bash
   git pull origin main
   chmod +x install-fixed.sh
   ./install-fixed.sh
   ```

3. **Restore data** (if needed):
   ```bash
   cp app.db.backup app.db
   ```

4. **Start services**:
   ```bash
   ./start-all.sh
   ```
